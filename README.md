## 🌟 Особенности

### 🎤 Голосовое управление
- Распознавание речи на русском (и автоопределение английского)
- Голосовые ответы вслух через `SpeechSynthesis`
- Режим **«Wake Word»** — активация по голосовой команде (можно настроить)
- Команда **«Стоп»** — прерывает ответ в любой момент

### 🧠 Умные ответы
- **Gemini API** (через Cloudflare Worker) — быстрые и точные ответы
- **Wikipedia** — автоматический резерв, если Gemini недоступен
- **Function calling** — погода, перевод, время в городах

### 📷 Распознавание фото
- Сделайте снимок через камеру и спросите: «Что это?»
- Gemini опишет изображение прямо в чате

### 🎬 Режим «Кинотеатр»
- Скрывает все лишние элементы интерфейса
- Остаётся только реактор и ответы — для полного погружения

### 📥 Экспорт диалога
- Сохраняйте историю чата в `.txt` файл одной кнопкой

### 💾 Сохранение истории
- Диалоги не исчезают после перезагрузки страницы

### 🎨 Стиль
- Арк-реактор с анимацией, частицами и радаром
- Космический фон, стеклянные панели, неоновые акценты
- Адаптация под телефон и планшет

## 🛠️ Как запустить

Это статичная HTML-страница — бэкенд не нужен.

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/kocmoc1337/Jarvis.git
   cd Jarvis
   ```
2. **Откройте файл `index.html`** в любом современном браузере.
3. **Разрешите доступ к микрофону** при первом клике на ядро.
4. **Начинайте говорить или печатать!**

> ⚠️ Голосовой ввод работает только в **Chrome, Edge или Safari 14.1+** и требует **HTTPS** (или `localhost`).

## 🛠️ Технологии

- **Чистый HTML/CSS/JS** — без зависимостей и сборщиков
- **Web Speech API** — голосовой ввод и вывод
- **Web Audio API** — звуковые эффекты и визуализация
- **Gemini API** (через Cloudflare Worker) — умные ответы
- **Открытые API** — Wikipedia, Open-Meteo (погода), MyMemory (перевод)
- **localStorage** — сохранение истории и заметок
