## 🌟 Особенности

### 🎤 Голосовое управление
- **Распознавание речи**: На русском языке через `SpeechRecognition API`.
- **Голосовые ответы**: J.A.R.V.I.S. говорит с вами вслух через `SpeechSynthesis API`.
- **Режим «Wake Word»**: Постоянно слушает и активируется по фразе «Джарвис» (можно настроить!).
- **Команда «Стоп»**: Прерывает ответ в любой момент.
- **Автоопределение языка**: Автоматически переключает распознавание между русским и английским.

### 🧠 Gemini AI (через Cloudflare Worker)
- **Безопасно**: API-ключ хранится в Cloudflare Worker, а не в коде.
- **Умные ответы**: Использует `gemini-2.0-flash` для естественных ответов.
- **Function calling**: Распознаёт намерения и вызывает функции (погода, перевод, время).
- **Обработка ошибок**: При исчерпании квоты автоматически переключается на Wikipedia.

### 📷 Распознавание фото
- **Камера**: Сделайте фото и спросите «Что это?» — Gemini опишет изображение.
- **Превью**: Фото отображается в интерфейсе перед отправкой.

### 🌐 Реальные данные (без API-ключей)
- **Википедия**: «Что такое чёрная дыра?», «Расскажи про Наполеона».
- **Погода**: По вашей геолокации (Open-Meteo).
- **Переводчик**: «Переведи слово мир на английский» (MyMemory).
- **Время в городах**: «Который час в Токио?».
- **Заметки**: «Запомни, что...» / «Что я просил запомнить?».
- **Таймер**: «Поставь таймер на 5 минут» + PWA-уведомление.
- **Контекст**: Понимает местоимения («А сколько ему лет?»).

### 🖥️ Текстовый ввод
- **Поле для ввода**: Альтернатива голосу, использует ту же логику поиска.

### ✨ Визуальный стиль
- **Арк-Реактор**: Светящееся ядро с частицами и радаром.
- **Космический фон**: Звёздное небо и туманность.
- **HUD-панели**: Стеклянные информационные панели (Диагностика, Время, Батарея).
- **Параллакс**: Реакция на движение мыши или наклон смартфона.
- **Потоковый вывод**: Ответы печатаются по буквам, как в настоящих чат-ботах.

## 🚀 Запуск

Это статичная HTML-страница, бэкенд не нужен.

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/kocmoc1337/Jarvis.git
    cd Jarvis
    ```
2.  **Откройте файл `index.html`** в вашем любимом браузере.
3.  **Разрешите доступ к микрофону** при первом клике на ядро.
4.  **Начинайте говорить или печатать!**

**Важно:** Голосовой ввод работает только в **Chrome, Edge или Safari 14.1+** и требует **HTTPS** (или `localhost`).

## 🛠️ Технологии

- **Чистый HTML/CSS/JS**: Никаких зависимостей и сборщиков.
- **Web Speech API**: Для распознавания и синтеза речи.
- **Web Audio API**: Для эмбиент-гула реактора и звуковых сигналов.
- **Gemini API (через Cloudflare Worker)**: Для умных ответов и распознавания фото.
- **Открытые API (без ключей)**:
    - Wikipedia REST API
    - Open-Meteo (погода)
    - MyMemory Translation
- **localStorage**: Для хранения заметок.
- **Service Worker (PWA)**: Для уведомлений таймера.
