# J.A.R.V.I.S. HUD

Голографический голосовой интерфейс в стиле «Железного человека» — работает прямо в браузере, без установки. Реагирует на голос, ищет реальную информацию в интернете и отвечает вслух.

---

## Возможности

**Голос**
- Распознавание речи на русском (`SpeechRecognition`), с автоперезапуском при тишине
- Голосовые ответы через `SpeechSynthesis`, с фиксами для Safari/iOS (там синтез речи иначе обрывается на длинных фразах)
- Wake word — режим постоянного прослушивания, реагирует на фразу «Джарвис»
- Команда «стоп» / «замолчи» прерывает ответ на середине

**Текст**
- Поле для ввода вопроса с клавиатуры — как альтернатива голосу, использует ту же логику ответов

**Ответы на реальные данные (без API-ключей)**
- Поиск в Википедии — «что такое чёрная дыра», «расскажи про Наполеона»
- Погода по геолокации (Open-Meteo)
- Перевод фраз (MyMemory) — «переведи слово мир на английский»
- Время в других городах — «который час в Токио»
- Заметки — «запомни, что...» / «что я просил запомнить»
- Таймер — «поставь таймер на 5 минут»
- Память контекста диалога — понимает местоимения («а сколько ему лет» после вопроса про человека)

**Визуал**
- Арк-реактор с частицами, радар-развёрткой и объёмным свечением
- Звёздное небо и туманность на фоне
- Стеклянные HUD-панели (диагностика, время, батарея, сеть)
- Параллакс ядра по движению мыши/пальца, наклон по гироскопу
- Глитч-эффект и звуковая вспышка при ошибках
- История запросов в выезжающей панели

## Как запустить

Это статичная HTML-страница, бэкенд не нужен.

1. Открой `index.html` в браузере (или ссылку на GitHub Pages, если включено)
2. Разреши доступ к микрофону при первом клике на ядро
3. Говори или печатай вопрос

**Важно:** голосовой ввод (`SpeechRecognition`) работает только в Chrome/Edge/Safari 14.1+, и требует https-контекста (не работает через `file://` или встроенные webview-превью).

## Технологии

- Чистый HTML/CSS/JS, без сборки и зависимостей
- Web Speech API (`SpeechRecognition` + `SpeechSynthesis`)
- Web Audio API — эмбиент-гул реактора, звуковые сигналы, реальный анализ микрофона для визуализации волны
- Открытые API без ключей: Wikipedia REST API, Open-Meteo, MyMemory Translation
- `localStorage` — хранение заметок между сессиями
- Battery Status API, Geolocation API, DeviceOrientation API
